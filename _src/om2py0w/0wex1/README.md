## 极简日记交互系统

###对象 
在Mac中运行[Diary-Mac.py](https://github.com/Iris-Di/OMOOC2py/blob/master/_src/om2py0w/0wex1/Diary-Mac.py)

在Windows中运行[Diary-Win.py](https://github.com/Iris-Di/OMOOC2py/blob/master/_src/om2py0w/0wex1/Diary-Win.py)

###使用：
1. 第一次运行程序，在程序所在文件夹创建diary.txt文件
2. 在“How are you today?"提问后，输入日记内容
3. 以后每次运行，添加一条日记所记录时间+日记，保存
4. 再次运行时，显示所有日记，并可以添加日记
